# `dateField(): ZodField<Date>`

A field to hold `Date` values.

</hgroup>

```ts
import { dateField } from "@form-atoms/field";

const joinedGameAt = dateField({
  required_error: "Please specify your birth date.",
});

const leftGameAt = dateField().optional();
```

Usable with [useDateFieldProps()](?path=/docs/hooks-useDateFieldProps--docs)

## Initial Config

| Param            | Value                 |
| ---------------- | --------------------- |
| `value`          | `undefined`           |
| `schema`         | `z.date()`            |
| `optionalSchema` | `z.date().optional()` |
