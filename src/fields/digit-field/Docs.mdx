# `digitField(): ZodField<0 | 1 | ... | 9 | undefined>`

A field to hold an integer - single digit - of type `number`.

</hgroup>

```ts
import { digitField } from "@form-atoms/field";

// Create 4 inputs, each accepting only number 0-9
const pinForm = formAtom({
  pinCode: [
    digitField({ name: "pin-1" }),
    digitField({ name: "pin-2" }),
    digitField({ name: "pin-3" }),
    digitField({ name: "pin-4" }),
  ],
});
```

## Initial Config

| Param            | Value                                         |
| ---------------- | --------------------------------------------- |
| `value`          | `undefined`                                   |
| `schema`         | `z.union([z.literal(0), ..., z.literal(9),])` |
| `optionalSchema` | `schema.optional()`                           |
