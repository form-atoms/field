# `arrayField<Value>(): ZodField<Value[]>`

A generic field to hold multiple values of the same type.

</hgroup>

```ts
import { arrayField } from "@form-atoms/field";
```

## Initial Config

| Param            | Value                            |
| ---------------- | -------------------------------- |
| `value`          | `[]`                             |
| `schema`         | `z.array(z.string()).nonempty()` |
| `optionalSchema` | `z.array(z.string())`            |

## Common cases

### `stringArrayField(): ZodField<string[]>`

Commonly used to capture answers from multi-choice components such as [CheckboxGroup](?path=/docs/components-checkboxgroup--docs) or [MultiSelect](?path=/docs/components-multiselect--docs).

```ts
import { stringArrayField } from "@form-atoms/field";
```
