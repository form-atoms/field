# FieldErrors

A JSX component version of the [`useFieldErrors()`](https://github.com/jaredLunde/form-atoms#usefielderrors) hook.

</hgroup>

````tsx
import { FieldErrors, textField } from "@form-atoms/field";

const firstName = textField();

const Example = () => (

{" "}
<FieldErrors field={firstName}>
  {({ errors }) => <div>{errors.join("\n")}</div>}
</FieldErrors>
); ```
