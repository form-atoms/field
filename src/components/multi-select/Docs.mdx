# MultiSelect

A combination of generic [useMultiSelectFieldProps](?path=/docs/hooks-usemultiselectfieldprops--docs) and [useSelectOptions](?path=/docs/hooks-useselectoptions--docs)
which renders the options as the children of `<select multiple>`.

</hgroup>

```ts
import { MultiSelect } from "@form-atoms/field";
```

### Alternative

- [CheckboxGroup](?path=/docs/components-checkboxgroup--docs) component is an alternative for multi-choice answer.
- Prefer the `MultiSelect` when there are more items to select from than fits the viewport, so users must scroll.

## Example

```tsx
import {
  Select,
  MultiSelectProps,
  FieldLabel,
  FieldErrors,
  ZodArrayField,
} from "@form-atoms/field";

const SelectField = <Option, Field extends ZodArrayField>({
  field,
  label,
  ...props
}: {
  label: ReactNode;
} & MultiSelectProps<Option, Field>) => (
  <div style={{ margin: "20px 0" }}>
    <FieldLabel field={field} label={label} />
    <MultiSelect {...props} field={field} />
    <FieldErrors field={field} />
  </div>
);
```
