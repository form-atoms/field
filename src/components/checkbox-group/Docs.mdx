# CheckboxGroup

A generic multi-choice component enabling users to select multiple values from a list of options.
It uses the [useOptions](?path=/docs/hooks-useoptions--docs) hook
which renders the options as a list of `<input type="checkbox">`.

</hgroup>

```ts
import { CheckboxGroup } from "@form-atoms/field";
```
